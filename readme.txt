== Description ==


Woocommerce Product Review Export/Import is a plugin for export and import review, rating and comments meta of woocommerce products . 

* One click export process
* Automatically add review meta
* Simple import process 
* Filter the reviews for non-existing products 


NOTE: This plugin require atleast Woocommerce v2.5.0.


== Installation ==

Upload the plugin, and just Activate it.
